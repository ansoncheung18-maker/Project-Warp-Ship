# Phase B：子系統設計 (Subsystem Design)
**專案**：曲率泡載人飛船 (Project Warp-Ship)  
**版本**：1.0  
**日期**：2026-07-23  
**作者**：Anson Cheung（14歲）  
**定位**：呢份文件將系統規格分解為 8 個子系統，定義每個子系統嘅功能、設計、介面同製造方案。


## 1. 子系統總覽

Warp-Ship 系統分為 **8 個主要子系統**：

| 編號 | 子系統 | 功能 | 關鍵技術 | 關聯 Project |
|:---|:---|:---|:---|:---|
| **1** | 推進系統 | 產生曲率泡，推動飛船 | Casimir 陣列、儲存環 | Project 8 |
| **2** | 偵測系統 | 偵測 0.5 光年內障礙物 | 微型化 LIGO | 新設計 |
| **3** | 導航系統 | 規劃航線、規避障礙 | AI 導航 | Project 6 |
| **4** | 通訊系統 | 飛船 ↔ 地球通訊 | 曲率泡通訊 | Project 9 |
| **5** | 生命維持系統 | 船員健康同生存 | 人工重力、生態系統 | 現有技術 |
| **6** | 輻射防護系統 | 屏蔽宇宙射線 | 水屏蔽層、鉛屏蔽 | 現有技術 |
| **7** | 能源管理系統 | 能量分配同儲存 | Dyson Swarm、儲存環 | Project 7 |
| **8** | 結構與居住系統 | 飛船框架、船員生活空間 | 模塊化設計 | 新設計 |

### 2.3 設計細節

| 組件 | 功能 | 技術來源 |
|:---|:---|:---|

```

↓
曲率泡形成 (< 0.01 秒)
↓
飛船以 10,000c 航行

```
